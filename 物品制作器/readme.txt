物品制作器 物品描述的json文件格式：

[
    {
        "ID": "stick",
        "名称": "我的名字\n可以换行！",
        "特殊值": 1,
        "标签属性": null
    },
    {
        "ID": "diamond",
        "名称": "§r§b魔力钻石\n§7被注魔的钻石， 可以用于合成物品。\n§9Botania",
        "特殊值": 1,
        "标签属性": {"can_place_on": {"blocks":["diamond_block"]}}
    },
]

把这个 json 文件放置在 插件数据文件/特殊物品制作器/ 目录下即可
在控制台输入 mkitem 选择此文件即可开始制作
