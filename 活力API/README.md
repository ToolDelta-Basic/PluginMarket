# 活力API

自动刷新 G79 Token

### 功能说明
- 定时调用 OpenAPI 接口刷新 Token。
- 支持启动时立即执行保活。
- 可自定义刷新频率。

### 配置项
配置文件通常位于插件的数据目录下，包含以下参数：

| 配置项 | 说明 | 默认值 |
| :--- | :--- | :--- |
| `OpenAPI地址` | 接口请求基础路径 | `https://nv1.nethard.pro/api/open-api` |
| `API密钥` | 必填，用于身份校验 | (无) |
| `调用方` | 接口调用标识 | `helperbot` |
| `自动刷新间隔` | 两次刷新的间隔时间（分钟） | `25` |
| `启用自动保活` | 是否开启定时任务 | `true` |
| `启动时立即刷新` | 插件加载后是否立即尝试刷新 | `true` |

### 使用方法
1. 下载该插件
2. 启动 ToolDelta 生成配置文件。
3. 在生成的配置文件中填写你的 `API密钥`。
4. 重启或热重载插件即可生效。
