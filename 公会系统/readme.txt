------------------------
【插件功能简介】
------------------------

本插件为 我的世界基岩版 服务器提供较为完善的公会系统，包括：

1. 公会创建与解散
   - 玩家可以花费钻石创建公会
   - 公会可随时解散，解散后所有成员退出

2. 公会加入与退出
   - 支持模糊搜索加入公会
   - 玩家可自由退出当前公会

3. 公会成员管理
   - 会长可设置职位、踢出成员、转让会长

4. 公会经验与等级(TODO)
   - 通过捐献物品获取经验，公会可升级
   - 经验溢出自动升级，多级连续升级

5. 公会据点
   - 公会可设置主据点
   - 成员可随时传送至据点

6. 公会聊天频道
   - 专属公会频道聊天
   - 默认频道为公共聊天

7. 公告系统
   - 会长可设置公会公告
   - 所有成员可查看

8. 效果增益系统
   - 公会可购买速度、力量、夜视等效果
   - 购买后给予全体公会成员
   - 支持不同等级，价格递增
   - 效果菜单中已购买等级高亮显示

9. 公会仓库 (TODO)
   - 未来版本将支持物品仓库存取

------------------------
【主要指令】
------------------------

公会      - 打开公会主菜单
gc [消息] - 在公会频道发送消息

------------------------
【使用说明】
------------------------

1. 在0.0.1 -> 0.1.0 变化较大，所以我们的数据文件已经从原来的"guilds.json"到"公会数据文件.json"
2. 其次，在我们的公会系统中，配置项全部在guild/config.py文件中，可修改的都做了注释
3. 开发中，不代表最终成品
4. 注意！如果直接将"guilds.json"直接复制到"公会数据文件.json"里面，可能会导致一些错误或者数据丢失，我们将会在下个版本修复
