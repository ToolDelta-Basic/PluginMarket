# QQLinker v1.7.0 — 模块开发指南

## 快速开始

### 最小模块

```python
from qqlinker_framework.core.module import Module
from qqlinker_framework.core.kernel.decorators import command, listen
from qqlinker_framework.core.kernel.events import GroupMessageEvent

class HelloModule(Module):
    name = "hello"
    version = (1, 0, 0)

    async def on_init(self):
        self.logger.info("Hello 模块已加载")

    @listen(GroupMessageEvent)
    async def on_message(self, event):
        if "hello" in event.message.lower():
            await self.send_group_msg(event.group_id, f"你好 {event.nickname}")
```

### 上传到模块市场

模块市场路径: `插件数据文件/模块源件/`

```bash
cp hello.py <框架数据目录>/插件数据文件/模块源件/
```

PluginMarket 接受的格式：`.py` / `.md` / `.txt` / `.json` / `.html`

## 事件系统

### v1.7.0 新架构：Lane Router

事件在诞生时声明 lane，不再手动指定事件类型字符串：

```python
from qqlinker_framework.core.kernel.events import (
    GroupMessageEvent,      # lane="chat"
    GameChatEvent,          # lane="realtime"
    PlayerJoinEvent,        # lane="realtime"
    AIResponseEvent,        # lane="ai"
    ConfigReloadEvent,      # lane="admin"
)

# 监听
@listen(GroupMessageEvent)
async def on_group_msg(self, event): ...

# 发布
from qqlinker_framework.core.kernel.events import SystemPanicEvent
await self.event_bus.publish(SystemPanicEvent(service="my_module"))
```

### 跨 Lane 通信

非 chat lane 的模块**不应该直接调用 `send_group_msg`**。通过 `OutboundMessageEvent` 发回 chat lane：

```python
from qqlinker_framework.core.kernel.outbound import OutboundMessageEvent

# AI 模块产出回复
outbound = OutboundMessageEvent(
    target_type="group",
    target_id=event.group_id,
    content="AI 回复内容",
    reply_to=event.event_id,
)
outbound.priority = 0  # 0=正常, 5=管理优先, -1=低优先
await self.event_bus.publish(outbound)
```

## 命令系统

### 基础命令

```python
@command(trigger=".help", description="显示帮助")
async def cmd_help(self, ctx):
    await ctx.reply("帮助信息...")
```

### 命令上下文 (CommandContext)

```python
ctx.user_id       # QQ 号
ctx.group_id      # 群号
ctx.nickname      # 昵称
ctx.message       # 原始消息
ctx.args          # 参数列表
ctx.mid           # 调用者权限层级
ctx.is_admin      # 是否为管理员
```

### 命令选项

```python
@command(
    trigger=".kick",
    description="踢出玩家",
    op_only=True,           # 仅管理员
    required_role="admin",  # 需要特定角色
    argument_hint="<玩家名>",
    cooldown=5.0,           # 冷却（秒）
    pipeline_topic="admin_commands",  # 管道分支
)
```

## 配置系统

### 模块配置节

```python
class MyModule(Module):
    name = "my_module"
    default_config = {
        "greeting": "欢迎！",
        "cooldown": 3.0,
    }

    async def on_init(self):
        greeting = self.config.get("greeting")
```

配置文件自动生成在 `配置/模块/my_module.json`。

### 配置热重载

```python
from qqlinker_framework.core.kernel.events import ConfigReloadEvent

@listen(ConfigReloadEvent)
async def on_config_reload(self, event):
    self.max_retry = self.config.get("max_retry", 3)
```

## 服务注入

声明 `required_services` 后框架自动注入：

```python
class MyModule(Module):
    required_services = ["dedup", "tool"]

    async def on_init(self):
        self.tool.register_tool({...})
        self.dedup.check("key")
```

服务不存在时模块自动跳过加载（非致命）。

## 定时任务

```python
from qqlinker_framework.core.module_helpers import ScheduledTask

class MyModule(Module):
    scheduled = [
        ScheduledTask(name="cleanup", interval=3600, callback="cleanup_old_data"),
    ]

    async def cleanup_old_data(self):
        self.logger.info("清理过期数据")
```

## Pipeline 分支

### 声明命令所属分支

```python
@command(trigger=".封禁", pipeline_topic="admin_commands")
async def cmd_ban(self, ctx): ...

@command(trigger=".AI", pipeline_topic="ai_processing")
async def cmd_ai(self, ctx): ...
```

### 内置分支

| 分支 | workers | timeout | 适用 |
|---|---|---|---|
| `admin_commands` | 2 | 10s | 管理命令 |
| `game_commands` | 4 | 5s | 游戏命令 |
| `ai_processing` | 2 | 120s | AI 处理 |
| `general` | 4 | 30s | 通用 |
| `background` | 1 | 60s | 后台任务 |

## 工具系统

```python
@tool(name="search", description="搜索网页")
async def tool_search(self, query: str) -> dict:
    result = await self.search_api(query)
    return {"results": result}
```

工具自动注册到 ToolManager，AI 模块可通过 `tool_mgr` 发现和调用。

## 调试

### 内置调试引擎

```python
@command(trigger=".debug events")
async def debug_events(self, ctx):
    from qqlinker_framework.services.debug_engine import DebugEngine
    stats = self.services.get("debug_engine").get_stats()
    await ctx.reply(str(stats))
```

### 日志

```python
self.logger.debug(...)    # DEBUG 级别
self.logger.info(...)     # INFO 级别
self.logger.warning(...)  # WARNING 级别
self.logger.error(...)    # ERROR 级别
```

日志文件: `日志/<模块名>.log`
