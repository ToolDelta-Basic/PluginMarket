# QQLinker v1.7.0 — 适配器开发说明

适配器是框架与外部平台之间的桥梁。新增平台支持只需实现 `IFrameworkAdapter` 接口并放置到 `adapters/` 目录。

---

## 适配器接口

```python
from qqlinker_framework.adapters.base import IFrameworkAdapter

class MyAdapter(IFrameworkAdapter):
    """自定义平台适配器"""
```

### 必须实现的方法

| 方法 | 说明 |
|---|---|
| `send_game_command(cmd)` | 发送游戏指令 |
| `send_game_message(target, text)` | 向游戏内目标发送消息 |
| `get_online_players()` | 获取在线玩家名列表 |
| `send_group_msg(group_id, msg)` | 发送群聊消息 → bool |
| `send_private_msg(user_id, msg)` | 发送私聊消息 → bool |
| `listen_game_chat(handler)` | 注册游戏聊天监听 → (player, msg) |
| `listen_group_message(handler)` | 注册群消息监听 → {user_id, group_id, nickname, message} |
| `listen_player_join(handler)` | 注册玩家加入监听 → player_name |
| `listen_player_leave(handler)` | 注册玩家离开监听 → player_name |
| `register_console_command(triggers, hint, usage, func)` | 注册控制台命令 |
| `get_plugin_api(name)` | 获取其他插件 API |
| `is_user_admin(user_id, config_mgr)` | 检查用户是否平台管理员 |
| `send_game_command_with_resp(cmd, timeout)` | 发指令并等响应文本 |
| `send_game_command_full(cmd, timeout)` | 发指令并等完整响应 |

### 可选实现的方法

| 方法 | 说明 |
|---|---|
| `listen_active(handler)` | 框架就绪 |
| `listen_frame_exit(handler)` | 框架退出 |
| `listen_player_pre_join(handler)` | 玩家预加入 |
| `send_game_title(target, text)` | 标题栏消息 |
| `send_game_subtitle(target, text)` | 小标题栏 |
| `send_game_actionbar(target, text)` | 行动栏消息 |
| `send_message_round_robin(group_id, msg)` | 轮询式群消息（多机器人） |
| `register_pre_plugin_api(name, min_version)` | 前置插件 API 注册 |
| `listen_dict_packet(packet_id, handler)` | 字典数据包监听 |
| `listen_bytes_packet(packet_id, handler)` | 二进制数据包监听 |
| `resolve_player_names(entries)` | UUID → 玩家名映射 |

## 适配器自动发现

v1.7.0 起，框架通过 Scanner 自动扫描 `adapters/` 目录：

```python
# adapters/ 下每个 `*_adapter.py` 文件都会被扫描
# 框架根据环境变量/配置自动选择合适的适配器
```

### 适配器注册

适配器通过 `ChannelHost` 初始化时注入：

```python
from qqlinker_framework.adapters.tooldelta_adapter import ToolDeltaAdapter
from qqlinker_framework.libraries.channel_host import ChannelHost

adapter = ToolDeltaAdapter(plugin_instance)
host = ChannelHost(adapter=adapter, data_path="./framework_data")
await host.start()
```

## 事件发布到 LaneRouter

适配器将外部事件转换为框架标准事件并发布到 LaneRouter：

```python
from qqlinker_framework.core.kernel.events import (
    GroupMessageEvent, GameChatEvent, PlayerJoinEvent,
)

class MyAdapter(IFrameworkAdapter):
    def listen_group_message(self, handler):
        async def _bridge(raw_msg):
            event = GroupMessageEvent(
                user_id=raw_msg["user_id"],
                group_id=raw_msg["group_id"],
                nickname=raw_msg["nickname"],
                message=raw_msg["message"],
            )
            await self.event_bus.publish(event)  # LaneRouter 自动路由到 chat lane
        self._raw_handler = _bridge
```

### Lane 映射

适配器只需发布事件实例，LaneRouter 从 `event.lane` 自动推断路由：

| 发布的事件类 | 路由到 |
|---|---|
| `GroupMessageEvent` | chat lane |
| `GameChatEvent` | realtime lane |
| `PlayerJoinEvent` | realtime lane |
| `SystemPanicEvent` | critical lane |

## 完整示例

```python
from qqlinker_framework.adapters.base import IFrameworkAdapter
from qqlinker_framework.core.kernel.events import (
    GroupMessageEvent, GameChatEvent, PlayerJoinEvent, PlayerLeaveEvent,
)

class MyPlatformAdapter(IFrameworkAdapter):
    """示例：接入自定义平台"""

    def __init__(self, platform_client, event_bus=None):
        self._client = platform_client
        self.event_bus = event_bus
        self._game_chat_handler = None
        self._group_msg_handler = None

    async def send_game_command(self, cmd: str):
        await self._client.execute(cmd)

    async def send_game_message(self, target: str, text: str):
        await self._client.tell(target, text)

    async def get_online_players(self):
        return await self._client.list_players()

    async def send_group_msg(self, group_id: int, message: str) -> bool:
        await self._client.send_message(group_id, message)
        return True

    async def send_private_msg(self, user_id: int, message: str) -> bool:
        await self._client.send_dm(user_id, message)
        return True

    def listen_game_chat(self, handler):
        async def _wrapper(player, msg):
            event = GameChatEvent(player_name=player, message=msg)
            if self.event_bus:
                await self.event_bus.publish(event)
        self._client.on_chat(_wrapper)

    def listen_group_message(self, handler):
        async def _wrapper(raw):
            event = GroupMessageEvent(
                user_id=raw.sender_id,
                group_id=raw.group_id,
                nickname=raw.sender_name,
                message=raw.text,
            )
            if self.event_bus:
                await self.event_bus.publish(event)
        self._client.on_group_msg(_wrapper)

    def listen_player_join(self, handler):
        async def _wrapper(name):
            event = PlayerJoinEvent(player_name=name)
            if self.event_bus:
                await self.event_bus.publish(event)
        self._client.on_join(_wrapper)

    def listen_player_leave(self, handler):
        async def _wrapper(name):
            event = PlayerLeaveEvent(player_name=name)
            if self.event_bus:
                await self.event_bus.publish(event)
        self._client.on_leave(_wrapper)

    def register_console_command(self, triggers, hint, usage, func):
        for trigger in triggers:
            self._client.register_cmd(trigger, func)

    def get_plugin_api(self, name):
        return None

    def is_user_admin(self, user_id, config_mgr) -> bool:
        return self._client.is_admin(user_id)

    async def send_game_command_with_resp(self, cmd, timeout=5.0):
        return await self._client.execute_and_read(cmd, timeout=timeout)

    async def send_game_command_full(self, cmd, timeout=5.0):
        return await self._client.execute_full(cmd, timeout=timeout)
```
