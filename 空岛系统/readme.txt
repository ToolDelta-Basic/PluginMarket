§b配置文件填写介绍：
§a"L型空岛"§r: {
    §b"结构名"§r: "空岛1",
    "介绍": "普通的L型空岛",
    §d"结构生成偏移"§r: [
        -9,
        -6,
        -4
    ]
}
§a"L型空岛"§r: 空岛包体名, 也就是 .is create 空岛包体名  用到的名字。
§b"结构名"§r: 用结构方块保存下来的空岛结构的名字。
§d"结构生成偏移"§r: 以玩家在空岛上着陆的坐标点为原点， 加载空岛结构的坐标的xyz偏移量。
§c重要！！！
  不推荐使用聊天栏菜单创建空岛， 你可以设置一个命令方块，
  里面写入 /execute as @p run w @a[tag=robot] .is create 所需的空岛包体名
  以激活命令方块来创建一个空岛。
  此功能需要配合插件 命令方块代发言支持 食用。
