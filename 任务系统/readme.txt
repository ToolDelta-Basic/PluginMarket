插件数据存放在 插件数据文件/任务系统/ 文件夹内。

任务数据的路径应当如下：

    插件数据文件/
        任务系统/
            任务组xxxx/
                任务1.json
                任务2.json
            yyyyyyyyy/
                任务1.json
                xxxx任务.json

任务的标签名就是 任务组文件夹名/任务文件名
